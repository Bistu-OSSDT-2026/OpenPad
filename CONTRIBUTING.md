# Contributing

## Branches

- `main`: stable submission branch.
- `dev`: integration branch.
- `feature/audio-pad`
- `feature/sample-chop`
- `feature/sequencer`
- `feature/gesture-fx`
- `feature/ui-architecture`

## Pull Requests

Target PRs to `dev`. Include:

- What changed.
- Which module changed.
- How to test.
- Whether `npm run build` passes.

## Commit Messages

Use specific messages, for example:

- `Add pad keyboard trigger`
- `Implement sample chop assignment`
- `Wire sequencer to triggerPad`
